## Descrição
O programa tem as funcionalidades base de um sistema bancario, como transferencias depositos e saques
atualmente possui 7 funcionalidades que sao
1. Sacar
2. Depositar
3. Transferencia
4. Pix
5. Consultar Saldo
6. Consultar Todos os dados
7. Registrar Cliente
8. Sair
